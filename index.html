Greetings! Welcome to my one-page resume site.

This page is designed in order for me to show off a few basic CSS 3 and jQuery skills. It is only supported in IE9+, and the latest Chrome and Firefox. Note that as this is my personal site, I only do a minimal amount of IE testing - too many things are broken in it, and given that I only spend my free time on this site, I try to dedicate said small amounts of free time to updating features/experience to match the newest browsers (it'd be a different story if I could get paid to update this site ;) )

The menu to the right is an unordered list (as in, no-link navigation!). Not that you can tell - just try clicking on any line to display the content there. If you wish to see this message again (for whatever reason - I can't imagine why you would) then just refresh the page.

This page currently uses the following technologies:

  • HTML 5 - not that this is significant, as I haven't included any real animated features that can't use jQuery
  • CSS 3 - that's how the navigation receives its box shadow!
  • jQuery - that's how the content is displayed
  • Adapt.js - try resizing your browser. Cool! The site will resize itself dynamically (or at least, download a new stylesheet to match the size of your browser). This isn't true responsive design, but it's close.
